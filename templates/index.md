# TPM Program Management Templates

## Available Templates

| Template | Description | Use Case |
|----------|-------------|----------|
| [Benefits Register](BenefitsRegisterTemplate.pdf) | Tracks benefits with baseline, target, actual values, owners, and status. | Benefit realization tracking |
| [Strategic Alignment Scorecard](StrategicAlignmentScorecard.pdf) | Scores program alignment with strategic objectives. | Program approval and prioritization |
| [Lessons Learned Log](ProgramLessonsLearnedLogTemplate.pdf) | Captures positive and negative lessons with action items. | Continuous improvement |
| [Program Charter](ProgramCharterTemplate.pdf) | Full program definition including vision, scope, risks, stakeholders. | Program initiation |

**How to use:** Download and adapt these templates for your specific programs. They follow best practices in Technical Program Management.
